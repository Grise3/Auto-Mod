How it works

AutoMod V2 is a tool focused on one thing: stopping compromised accounts from damaging your server. It uses three simple methods to detect suspect behavior.

🛡️ Pattern Detection

This module watches for users sending the exact same message across different channels in a short time. This is a common sign of a hijacked account or a basic bot.

If a user hits the limit you set, the bot will automatically apply the configured sanction (like a 24h timeout).

🔗 Link Filtering

Hacked accounts often post server invites to spread further. This module scans messages for Discord invite links.

You can choose to block all invites, or only those that contain suspect keywords (like "nitro", "gift", or "free") often used in scams.

🖼️ Image Similarity Detection

Some scams use images to bypass text filters. The bot uses a two-step detection system:

1. Perceptual Hash (pHash): A fast fingerprint that detects visually identical images even if resized or compressed.

2. SSIM (Structural Similarity): Measures perceptual differences by comparing luminance, contrast, and structural patterns.

You can configure the similarity threshold per server.

⚙️ Commands

/config panel

Opens the interactive dashboard to manage status, sanctions, and immune roles/channels.

/config getcode

Generates a 15-character seed representing your current logic settings.

/config setcode

Applies a configuration instantly using a seed code from our online tool.

/stats

Shows how many times the bot has intervened on your server.
